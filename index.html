Face Recognition Project

Face Recognition | Machine Learning Project(Self)

May 2023

Platform- Pycharm |Language- Python |

This project combines the Haar cascade classifier and LBPH (Local Binary Patterns Histograms) Face recognizer to create a face recognition system. The goal is to greet the user with a personalized message if a face is successfully detected and recognized with a confidence level above 65. If a face is not identified, the system will audibly announce "Face not found".In cases where no face is detected, the system audibly notifies the user with the message "Face not found". This ensures clear communication when face detection fails.
